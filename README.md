# وب‌سایت لندینگ در دسکتاپ

یک وب‌سایت تک‌صفحه‌ای مدرن ساخته‌شده با HTML، CSS و JavaScript خالص.
این قالب یک صفحه لندینگ است با طراحی ساده و مدرن، ناوبری روان. شامل بخش‌های اصلی یک لندینگ شرکتی مانند: هدر (Hero)، خدمات، درباره ما، نمونه‌کارها، پلن‌های قیمت، نظرات کاربران و فرم تماس است.

## تکنولوژی‌ها

- HTML5 معنایی
- CSS3 با استفاده از Grid و Flexbox
- JavaScript خالص برای منوی موبایل و اسکرول نرم
